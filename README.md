### 🌐 About the Project

This is my personal portfolio website, a showcase of my skills as a **junior web developer**.  
It highlights responsive design, clean code structure, interactive elements, and modern UI practices using only vanilla technologies (no frameworks yet , keeping it lightweight and focused on fundamentals).

**Core Goals of this site:**
- Demonstrate strong HTML5 semantic structure & accessibility basics
- Show responsive layouts that work beautifully on mobile, tablet, and desktop
- Include smooth JavaScript interactions (e.g., navigation, animations, form handling)
- Serve as a living resume for job applications and freelance inquiries
